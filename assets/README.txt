CrossFit OTL Consultation Tool · Asset Folder
=============================================

TWO APPS IN THE CROSSFIT OTL TECH STACK:

  1. PUSHPRESS (industry standard for CrossFit affiliates)
     · Class booking, attendance, check-ins
     · OTL Committed Club streak tracker
     · Billing / membership management

  2. ON THE LINE FITNESS (white-label of Trainerize)
     · Lifestyle & nutrition coaching
     · Meal planning (6-day rotation)
     · Food logging + barcode scan + step counting
     · Habit tracking, stats, badges
     · App Store: apps.apple.com/us/app/crossfit-otl/id6473308995

HOW TO GET SCREENSHOTS ONTO DISK
=================================
  Option A - AirDrop:
    iPhone Photos -> Select -> Share -> AirDrop -> This Mac
    Files land in ~/Downloads/
    Rename to match the filenames above
    Move into this folder

  Option B - iCloud Photos on Mac:
    Mac Photos app -> find the screenshots -> drag to Finder window

  Option C - Message to self:
    Send screenshots via iMessage to yourself
    Right-click in Messages app on Mac -> Save As
